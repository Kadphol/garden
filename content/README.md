# Content Vault

This directory is the Markdown source of truth for the site.

- `garden/`: evergreen and evolving notes
- `posts/`: dated articles and essays
- `projects/`: portfolio case studies
- `pages/`: standalone pages such as About

Only entries with `published: true` are included in generated routes, search,
feeds, tags, related content, and backlinks. Use the frontmatter example in the
root `README.md` when creating the first entry.
